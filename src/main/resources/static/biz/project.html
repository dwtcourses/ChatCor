프로젝트 생성

프로젝트란 하나의 챗봇에 적용되는 데이터 세트를 의미합니다. 프로젝트를 생성하면 시스템에는 프로젝트의 이름과 함께 네 개의 새로운 폴더가 만들어 집니다.

  • intent : 인텐트 결과값
  • entity : 엔트리 값들로 합쳐진 엔티티값
  • entry : 사용자가 임의로 올리는 용어
  • sentence : 사용자가 임의로 등록하는 문장

분석, 정제는 sentence 폴더에 저장된 문장을 중심으로 처리 됩니다.

Entity 편집은 entry 폴더에 올라온 파일 내에 나열된 데이터를 중심으로 entity에 값을 저장합니다.
